Python知识点（三）

发表于 2020-04-09 | 分类于 python
| 字数统计: 793 字 | 阅读时长 ≈ 2 分钟

摘要:浅谈Python基础知识点…

1、python的可变类型与不可变类型有哪些？

​ ①、可变类型：列表、字典、可变集合（set）

​ ②、不可变类型：整型、浮点型、复数、字符串、元组、不可变集合（frozenset）

2、python中深拷贝和浅拷贝的区别?

​ ①、深拷贝：改变引用对象的值，被引用对象的值也会改变。

​ 例如“=”，“deepcopy（）”

​ ②、浅拷贝：改变引用对象的值，被引用对象的值不会改变。

​ 例如“copy（）”，切片

3、python的设计模式有哪些？

​ ①、单例模式：保证一个类仅有一个实例，并提供一个访问他的全局访问点。

​ ②、装饰器模式：不修改元类代码和继承的情况下动态 扩展类的功能。

​ ③、迭代模式：提供一个方法顺序访问聚合对象中各个元素。

​ ④、命令模式：将“请求”封装层对象，以便使用不同的请求，队列或者日志来参数化其他对象。命令模式也支 持撤销的操作。

4、列表推导式和生成器表达式的优劣？

​ ①、[item for item in iterable]列表推导式是将所有的值一次加载到内存中。

​ ②、(item for item in iterable)在大数据量处理时，生成器表达式的优势就体现出来了，因为它的内存使用方 更好、效率更高。它并不创建一个列表，只是返回一个生成器。

5、装饰期实现单例模式。
6、使用装饰器的单例模式和使用其他方法的单例，在后续使用中有什么区别?

​ 使用装饰器单例属性不会被覆盖。因为装饰器单例模式是直接返回之前生成的对象，并不会重新初始化。

7、什么是函数闭包？

​ 函数闭包指函数中嵌套函数且外层函数的返回值也是函数，闭包中需要注意的是变量的作用域。

8、什么是元类？

​ 元类就是负责帮你创建类的类，object。

9、python的GIL。

​ GIL是python的全局解释器锁。同一个进程中加入有多个线程运行，一个线程在运行python程序的时候会占用python的解释器，使该进程内的线程无法运行，等该线程运行完后其他线程才能运行。如果线程运行中遇到耗时操作，则解释器锁解开，使其他线程运行。所以在多线程中，线程的运行仍是有先后顺序的，并不是同时进行。

10、python退出时，为什么不是所有的内存都被解除分配？

​ 每当python退出时，尤其是那些对其他对象具有循环引用的python模块或者从全局名称空间引用的对象并不总是被解除分配或者释放。由于python拥有自己的高效清理机制，无法解除分配保留的那些内存部分会在退出时尝试取消分配/销毁其他所有对象。
