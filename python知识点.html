1、if __name__ == '__main__'有什么作用？

__name__是当前模块名，当模块被直接运行时模块名为 __main__。既当模块直接运行时，if __name__ == '__main__'下的代码会被执行，而当模块是被作为包导入时，则if __name__ == '__main__'的代码不被运行。

5、__init__.py的作用？

它是作为包的一部分，标识包。

__init__.py 作为标识 Python 包的文件, 这么做是为了防止 Python 在引入 module 时误引入其他类似 string 这样的通用名. __init__.py 最简单的方式是空文件, 但是它可以为包执行初始化代码或者设置 __all__ 变量。
